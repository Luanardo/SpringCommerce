# SpringCommerce
## Tổng quan dự án
- Dự án này sử dụng thiết kế hướng đối tượng để tạo các lớp và đối tượng có ý nghĩa và tái sử dụng được.
- Sử dụng mô hình MVC để tách biệt giữa logic dữ liệu (Model), hiển thị (View), và điều khiển (Controller).
